Eccentricity-Driven Error Scaling in the Kepler ProblemThis repository contains the Python implementation and datasets for a comparative scaling analysis of numerical integrators in orbital mechanics. Project OverviewWe investigate the secular growth of energy error in the unperturbed Kepler two-body problem as a function of orbital eccentricity $e$. The study benchmarks three integration schemes across $e \in [0.1, 0.7]$ to quantify their mathematical sensitivity to orbital geometry. Key FindingsPower-Law Scaling: Fractional energy error obeys strict power-law scaling of the form $|\Delta E / E_0| [cite_start]\propto (1 - e)^{-k}$. Exponent Stratification: Scaling exponents span nearly an order of magnitude, from $k \approx 0.44$ (Euler) to $k \approx 8.61$ (RK4). Symplectic Advantage: The Velocity Verlet integrator exhibits zero net secular drift, preserving angular momentum to machine precision ($\sim 10^{-15}$). Phase-Shock Discovery: First-order methods suffer a ~25% sensitivity to starting location (periapsis vs. apoapsis), a phenomenon absent in higher-order and symplectic schemes. RequirementsPrecision: All simulations require strict 64-bit double-precision arithmetic (\texttt{float64}) to prevent hardware noise floors from masking algorithmic truncation error.
